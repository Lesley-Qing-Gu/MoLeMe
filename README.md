
# 🚀 摸了么 (MoLeMe) - 职场救赎终端

**“与其内卷自我，不如带薪摸鱼。”**

`摸了么` 是一款专为当代打工人设计的反向打卡与摸鱼监测系统。它采用复古的 Windows 98 UI 风格，集成了多种“带薪生存”工具，旨在帮助用户在枯燥的办公环境中寻找心理慰藉，量化摸鱼收益，提升职场幸福感（而非生产力）。

---

## 🛠️ 核心功能

### 1. 💻 我的电脑 (带薪收益监视)
*   **实时计费**：输入你的日薪，系统将自动换算成秒薪。
*   **收益量化**：精准记录你每一秒摸鱼为公司带来的“亏损”和你个人的“净利润”。
*   **段位系统**：从“职场牛马”逐步升级至“摸鱼至尊”，记录你的救赎之路。

### 2. 🗂️ 我的文件 (生存任务清单)
*   **每日目标**：提供一系列心理减压任务。
*   **进度追踪**：完成任务（如敲木鱼、假装更新、乱涂乱画）即可获得成就感奖励。

### 3. 📧 我的邮箱 (电子木鱼 2.0)
*   **拟真体验**：基于实物照片建模的梨形木鱼，配有红黄相间的锦缎坐垫。
*   **功德积累**：每一次敲击都会弹出“功德+1”，消除被老板 PUA 产生的业障。
*   **心中无 KPI**：专注敲击，让心灵在喧嚣的办公室获得片刻宁静。

### 4. 🗑️ 回收站 (垃圾话处理中心)
*   **匿名吐槽**：在这里输入你对加班、KPI 或老板的不满。
*   **物理释放**：点击“揉成纸团丢掉”，你的负能量将被瞬间揉碎并精准投入二次元垃圾桶。

### 5. 🚨 老板雷达 (Panic Mode)
*   **一键应急**：鼠标悬停在右下角红色的“老板来了”按钮上，屏幕瞬间切换至高大上的“Q1 业务增长分析报告”。
*   **拟真假数据**：充满各种复杂图表和专业术语（如：成本坍缩、边际效用递减），让你的摸鱼行为在 0.1 秒内变得极其专业。

### 6. ⏳ 假装系统更新
*   **硬核掩护**：模拟 Windows 10/11 的蓝色配置更新界面。
*   **99% 的艺术**：进度条会在 99% 处故意停留，为你争取完美的接咖啡或去洗手间的时间。

---

## 🎨 设计理念

*   **视觉风格**：致敬 Windows 98。经典青色桌面、灰色窗口边框、像素化图标，勾起你对那个“还没开始卷”的时代的记忆。
*   **交互逻辑**：采用经典的 `Start` 按钮作为摸鱼计时的开关。按下是“救赎”，弹起是“生存”。
*   **情感连接**：通过幽默、讽刺的文案，缓解职场焦虑，提醒每一个打工人：命是自己的，钱是公司的。

---

## 💻 技术栈

*   **框架**：React 19
*   **样式**：Tailwind CSS (定制 Win98 边框样式)
*   **动画**：CSS3 Keyframes & SVG 滤镜
*   **状态管理**：React Hooks & LocalStorage 持久化
